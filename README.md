#  SafeZone - Annual Trip Management System
מערכת חכמה למעקב ובקרה בזמן אמת לטיולים שנתיים.

מערכת SafeZone מספקת פתרון אבטחה טכנולוגי המאפשר איתור מקום בזמן אמת, זיהוי מיקומים גאוגרפיים של תלמידות לפי שיוך למורה, וניטור חריגות מטווח בטיחות מוגדר.


## תכונות עיקריות
*  ניהול משתמשים: רישום והתחברות מאובטחת למורות באמצעות תעודת זהות ושימוש ב-Spring Security.
*   מעקב בזמן אמת: הצגת מפה דינמית הכוללת את מיקומי המורה וכל תלמידות הכיתה בו-זמנית.
*   חישוב מרחק חכם: שימוש בנוסחת האברסין  לחישוב מרחק מדויק על פני כדור הארץ.
*   התרעות ויזואליות: זיהוי אוטומטי וסימון תלמידות שחרגו מטווח של **3 ק"מ** מהמורה בצבע אדום.
*   ניהול כיתה: ממשק נוח להוספת תלמידות חדשות, צפייה ברשימות כיתתיות וניהול נתוני כלל בית הספר.


##  טכנולוגיות

### Backend (Java Spring Boot)
*   **Spring Security:** אבטחת ה-API והרשאות מבוססות תפקיד.
*   **REST API:** חשיפת נקודות קצה לתקשורת עם צד הלקוח.
*   **Haversine Formula:** אלגוריתם לחישוב מרחקים גאוגרפיים.

### Frontend (React)
*   **React Leaflet:** ספריית מפות אינטראקטיבית.
*   **Axios:** ביצוע קריאות HTTP לשרת.
*   **React Router:** ניווט בין דפי המערכת (Dashboard, Login, Register).
*   **Local Storage:** שמירת מצב התחברות בדפדפן.


## מבנה הפרויקט

###  Backend / Java
*   `SecurityConfig` - הגדרות אבטחה, CORS וסינון גישה לנתיבים.
*   `StudentController` & `TeacherController` - ניהול הבקשות הנכנסות עבור תלמידות ומורות.
*   `StudentService` & `TeacherService` - לוגיקה עסקית, עיבוד נתוני מיקום וחישובי מרחק.
*   `UserDetailsService` - בשימוש עם Spring Security לניהול הזדהות מורות.

###  Frontend / React
*   `TeacherDashboard` - דף הבית המרכזי המשלב מפה, ניהול כיתה והוספת תלמידות.
*   `MapPage` - רכיב המפה המבצע עדכוני מיקום אוטומטיים וזיהוי חריגות.
*   `TeacherLogin` / `TeacherRegister` - דפי הזדהות ורישום למערכת.


## הוראות הרצה

### צד שרת (Backend)
1. יש להפעיל את פרויקט ה-Spring Boot.
2. השרת ירוץ כברירת מחדל בכתובת: `http://localhost:8080`.

### צד לקוח (Frontend)
1. יש להתקין את התלויות  npm install
